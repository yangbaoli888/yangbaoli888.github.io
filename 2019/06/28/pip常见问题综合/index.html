一、pip常用安装命令

  1. 安装指定版本 pip install tensorflow==1.2.0
  2. 升级到最新版本 pip install –upgrade tensorflow
  3. 卸载 pip uninstall TensorFlow -y
  4. 卸载指定版本 pip uninstall TensorFlow==1.2.0 -y
  5. 获取使用pip已安装文件 pip list install
  6. 指定安装路径 pip install filename --target=path 绝对路径
